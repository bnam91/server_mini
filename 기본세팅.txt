# git clone 이후 생성할 파일 2가지

- auth경로.txt
- ID.txt

=============================================

1. auth.경로.txt
- github > api_key > auth.py의 경로 선택

2. ID.txt
- 구글 시트명과 동일하게 입력
- (예시) 매칭할 구글시트명 'mini_01'이라면 ID.txt에도 'mini_01'입력
